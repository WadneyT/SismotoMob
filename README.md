# sismotoMOB

Aplicativo mobile em Expo SDK 54 para cadastro e listagem de produtos, clientes e pedidos.

## Como rodar

Instale as dependencias:

```bash
npm install
```

Inicie o Expo:

```bash
npm start
```

Depois, abra o QR Code no Expo Go.

## API

A URL base da API fica em:

```txt
src/servicos/api.js
```

Se estiver usando ngrok, atualize o `baseURL` sempre que o tunel mudar.
